Borrowing an item

Click the item you want on the Home page

Now, you can see the detailed information of the selected item


Check item Information

Check the description in Basic Info tab

Check Pricing and Condition

Check Address

 

Check Reviews

Check Reviews

Click "Borrow" if you want to borrow this item


Enter Borrow Information

Check item information again


 

Enter borrowing dates (start / end)

Click "Preview" if you want to borrow the selected item

   

Check Borrowing Information

Check borrowing dates (start / end)

Check borrowing days, total fee, and deposit

Click "Borrow" if information is correct

Your request will be sent to the item owner

 

Check the reservation is confirmed

You can check the reservation process in the My Space 

 

Click My Borrow

In Requested tab, you can check current requested item for borrowing

You can see the requested item in the Borrowing tab if the item owner already confirmed your request

If you want to cancel your request, click "Cancel Request"

 

Request confirmation of return

Click Borrowing tab

Click "Request Return" if you want to return an item, item status will change to RequestReturn

Request Return will be sent to the owner

You can cancel the reservation if borrowing has not already started


 

Check return completed

Click Completed tab

Check Refund Deposit is correct

 

 

Review an item

Click "Review" if you want to write a review

Select Stars, Write a title and review
